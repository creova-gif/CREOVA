## 🎯 HOW TO USE THE MULTILINGUAL SYSTEM

### **For Content Updates:**

**Adding New Translation:**
1. Open `/context/LanguageContext.tsx`
2. Add key to `translations.en` object:
   ```typescript
   'section.key': 'English Text'
   ```
3. Add French translation to `translations.fr` object:
   ```typescript
   'section.key': 'Texte Français'
   ```
4. Use in component:
   ```typescript
   const { t } = useLanguage();
   <p>{t('section.key')}</p>
   ```

### **For New Pages:**

1. Import `useLanguage` hook:
   ```typescript
   import { useLanguage } from '../context/LanguageContext';
   ```

2. Use `t()` function in component:
   ```typescript
   const { t } = useLanguage();
   ```

3. Replace hardcoded text:
   ```typescript
   // Before:
   <h1>Welcome</h1>
   
   // After:
   <h1>{t('page.welcome')}</h1>
   ```

4. Add translation keys to LanguageContext

### **For Testing:**

1. Click EN/FR toggle in header
2. Verify all text updates instantly
3. Check for overflow on mobile
4. Test booking/checkout flow
5. Verify forms work in both languages
